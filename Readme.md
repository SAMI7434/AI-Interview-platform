🤖 AI-Powered Interview Platform

An AI-driven mock interview platform that simulates real interview experiences using advanced AI services. It integrates speech recognition, intelligent question generation, and real-time interaction to help users prepare effectively.

Built using the MERN stack and deployed with a serverless architecture for scalability and cost efficiency.

🚀 Features
🎯 AI-Based Question Generation
Generates role-specific interview questions dynamically using Gemini AI
🎤 Real-Time Speech-to-Text
Converts spoken answers into text using Azure AI Speech Services
⚡ Scalable Serverless Backend
Powered by AWS Lambda and API Gateway
💻 Interactive Coding Environment
Supports live coding during interviews
🔄 Real-Time Communication
Enables seamless interaction using Socket.io


🛠️ Tech Stack



🌐 Frontend (Client)
React.js + Vite – Fast and modern UI development
Tailwind CSS – Responsive and utility-first styling
Lucide React – Clean and customizable icons



⚙️ Backend (Server)
Node.js & Express.js – REST API development
MongoDB Atlas – Cloud-based database
Socket.io – Real-time communication
Azure AI Speech-to-Text – Voice transcription
Gemini AI – AI-based question generation




☁️ Deployment & Infrastructure
AWS Lambda – Serverless compute
API Gateway – API management
Amazon S3 – Storage for assets
Netlify – Frontend hosting
